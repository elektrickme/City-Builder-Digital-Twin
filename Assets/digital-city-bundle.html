Level State

Generate Random picks a new seed on every click and rebuilds obstacles and centers using the same 25px procedural cell size and placement rules as the embedded game's auto map (then sets grid size to 25 so paint snap matches). The field shows the seed for the layout on screen, and the same value is stored in JSON to replay that map. New seed only replaces the stored seed without changing geometry.
